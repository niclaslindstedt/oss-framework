title: Components module
---

A `components` module of shared UI primitives — `Modal`, `Button`, `Checkbox`, `ClearableInput`, the `SelectPicker` dropdown, the `FloatingPanel` / `DismissBackdrop` / `useFloatingPosition` floating-panel kit, and a dependency-free inline glyph set — all painted through the theme token vocabulary and free of i18n, domain types, and asset imports.
