title: Copy-to-clipboard button
---

`components`: `CopyButton` — a bordered glyph button that drops a value (a string or a sync/async getter) on the clipboard and flashes a tick, backed by the new `useClipboard` hook and pure `copyTextToClipboard` helper (`hooks`) that own the robust write (Clipboard API with an `execCommand` fallback) and the self-resetting "copied" flag.
