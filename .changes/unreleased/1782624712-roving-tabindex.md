---
type: Added
title: Roving tabindex hooks
---

`useRovingTabindex` / `useGridRovingTabindex` add WAI-ARIA roving-tabindex keyboard navigation (one Tab stop, arrow keys move within the widget, Home/End jump to the ends, optional type-ahead) for listboxes, radiogroups, menus, and grid pickers — now wired into the `glyphs` `GlyphPicker` and `ColorPalette` so both are arrow-key navigable.
