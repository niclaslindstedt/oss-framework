type: Added
title: Live theme preview gallery
---

`theme`: a new `ThemePreview` component renders a compact, inline-styled miniature of the chrome straight from a palette — independent of the host's Tailwind tokens or the applied theme, so it previews any palette anywhere. `AppearancePicker` now uses it: the theme variant picker is a gallery of live preview cards (themes told apart by how they look, not by name), and Custom mode shows a live preview that tracks your colour edits.
