title: Sync module
---

A `sync` module: `SyncStatus` is a header glyph that morphs over a `SaveStatus` union (plus `dirty` / `offline`), and `SyncDetailsModal` is the command centre that spells out what sync is doing and why a save failed, with Save now / Reconnect / Reload / Check connection actions, a backend / encryption / file-location grid, and an optional collapsible developer log — all purely presentational over your app's own sync engine.
