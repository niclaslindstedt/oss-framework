title: useMediaQuery / useDesktopPointer
---

A `useMediaQuery` hook (subscribes to a CSS media query, reading the initial match synchronously to avoid a first-paint flash) and a `useDesktopPointer` helper for the precise-hovering-pointer query; `Checklist` gains an `onRowContextMenu` prop so a desktop right-click can reach a row's actions the way a touch swipe does.
